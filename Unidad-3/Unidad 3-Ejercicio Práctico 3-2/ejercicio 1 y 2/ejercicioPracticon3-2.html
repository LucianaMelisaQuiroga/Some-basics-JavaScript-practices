  1. Ejercicio 1: Definición de Algoritmo (dar un ejemplo):

    2. Es un conjunto de instrucciones, ordenadas lógicamente de manera que, ejecutándolas en ese orden, se llegue al resultado esperado para resolver/solucionar un problema. Precede al código en lenguaje de programación. Con el pensamiento algorítmico nos planteamos el problema y pensamos la solución. Se pueden expresar con diagramas de flujo. Tiene un inicio, input (datos de entrada), proceso/s, output (resultado) y un fin.

  2. Ejercicio 2: Qué es Pseudocódigo (dar un ejemplo):

    3. Es el lenguaje de descripción de un algoritmo, expresando en forma detallada cuáles son los pasos que seguirá el programa, pero en lenguaje natural. No existe una forma estándar de escribirlo. Se lo utiliza una vez definido el algoritmo (es decir, una vez que se identificó el problema cómo solucionarlo, o cómo llegar al resultado esperado), y antes de pasarlo a código.

    Ejemplo de pseudocódigo

    Pseudocódigo de un programa que pide 2 números, los suma, y muestra el resultado:

    • Inicio
    • Mostrar "Escribe el primer número"
    • Leer num1
    • Mostrar "Escribe el segundo número"
    • Leer num2
    • Sumar num1 + num2 y guardar en variable resultadoSuma
    • Mostrar resultadoSuma
    • Fin
